## 결론

`가려듣다`, `덧싣다`, `쏟아붓다`, `흘려듣다`를 core 불규칙 예외로 승격한다. 네 항목은
한국어기초사전 활용 record가 없어 자동 enriched 승격 조건을 충족하지 않지만, 고정한
표준국어대사전과 우리말샘 snapshot이 같은 진단형을 지지하고 충돌하는 규칙형 record가 없다.
각 활용과 오활용은 별도 fixture로 함께 검증한다.

자동 승격 조건은 바꾸지 않는다. enriched 배포 행은 그대로이고 생성 report에서 네 항목만
`review`에서 `core-duplicate`로 이동했다. 전체 `core_duplicate_count`는 40에서 44,
`review_count`는 418에서 414가 됐다.
